- [open] - Blue background (#3B82F6)
- [planned] - Purple background (#A855F7)
- [in-progress] - Orange background (#FB923C)
- [shipped] - Green background (#22C55E)
- [rejected] - Red background (#EF4444)

### Type Badges:
- [Bug] - Red background (#EF4444)
- [Feature] - Green background (#22C55E)

## Android Layout Differences

The Android version follows the same structure but with Material Design components:
- Uses native Fragment navigation instead of XAML Pages
- RecyclerView instead of ItemsControl
- MaterialCardView with 12dp elevation
- Bottom sheets for certain interactions
- Native Android Material buttons and text fields
- Programmatically created layouts (no XML)

## Responsive Behavior

### Windows (WinUI):
- Cards resize fluidly with window
- Horizontal scrolling for long content
- Hover effects on cards
- Smooth animations

### Android:
- Vertical scrolling via RecyclerView
- Touch ripple effects
- Swipe-to-refresh (can be added)
- Bottom navigation integration

## Accessibility Features

- Screen reader support via AutomationProperties (WinUI)
- ContentDescription (Android)
- Keyboard navigation
- High contrast mode support
- Proper focus indicators
- Semantic labels
